Intermediate Challenge

Task: Write a program to check if a number is even or odd.

number = int(input("Enter a number: "))

if number % 2 == 0:
    print("Even")
else:
    print("Odd")

Advanced Challenge

Task: Create a function that finds the largest number in a list.

def largest_number(numbers):
    return max(numbers)

print(largest_number([4, 9, 2, 15, 7]))

Python Coding Tips

  • Read the problem carefully before writing code.
  • Break the problem into smaller steps.
  • Use meaningful variable names.
  • Test your code with different inputs.
  • Debug errors by checking one line at a time.
